# SA-CRIMINAL-DATA
The South Africa Criminal Database project is a structured SQL  designed to simulate real-world crime and financial investigation data analysis.
This project contains 30 carefully designed SQL questions that test fundamental to intermediate querying skills such as filtering, aggregation, grouping, sorting, and analytical reporting.
 Objectives:
The goal of this project is to Improve problem-solving with real-world scenarios, Practice data analysis using structured datasets
🔹 Basic Retrieval
Display all records
Filter by crime type
Retrieve arrested suspects
🔹 Aggregations
Count total cases
Count cases per province
Count cases per crime type
Compare arrested vs non-arrested suspects
🔹 Financial Analysis
verage financial score
Highest estimated fraud amount
Top 10 fraud cases
Cases grouped by associated bank

How to Use This Project
Import the dataset into:
SQL Server
MySQL
PostgreSQL
Any relational database system
Run the SQL questions in your query editor.
Attempt solving and find solutions.
